# Render_Sprint_7_TripleTen

Este proyecto consiste en una aplicación web desarrollada con **Streamlit** que permite realizar un análisis exploratorio interactivo de un conjunto de datos de anuncios de venta de vehículos en Estados Unidos.

La aplicación tiene como objetivo facilitar la exploración visual de los datos, permitiendo identificar patrones y relaciones entre variables de forma sencilla e intuitiva.

---

## Funcionalidades principales

La aplicación permite al usuario:

- Visualizar la **distribución del kilometraje** de los vehículos mediante un histograma interactivo.
- Analizar la **relación entre el precio y el kilometraje** a través de un gráfico de dispersión.
- Interactuar con las visualizaciones usando casillas de verificación, sin necesidad de ejecutar código.

Todas las visualizaciones fueron creadas utilizando **Plotly Express**, lo que permite una experiencia interactiva dentro del navegador.
